# Python学习资源推荐系统 python1113

## 项目概述

本项目是一个基于Python技术的学习资源推荐系统，致力于为用户提供优质的学习资源，并通过高效的后台管理系统实现对资源的维护与更新。

## 技术栈

- 后端：Python + Django
- 前端：Vue
- 数据库：MySQL

## 系统角色

- 管理员：负责系统的管理与资源更新
- 用户：使用系统进行学习资源查找和获取推荐

## 系统特点

- **前后分离**：本项目采用前后端分离的设计，前端使用Vue技术，后端使用Python和Django框架。
- **详尽的文档**：包含万字文档，提供visio图表模板，涵盖用例图、ER图、结构图等多种图表。
- **开发工具提供**：提供完成项目所需的所有软件工具。
- **调试教程与视频**：提供视频调试教程以及同类技术开发项目的核心技术讲解。
- **图文修改教程资料**：提供详细的注册登录背景图、标题等修改教程。

## 运行环境

- 确保电脑配置的Node版本高于14，以适应远程配置好的cnpm环境。

## 部署步骤

由于项目的具体部署步骤涉及具体操作，以下仅为概述：

1. 准备技术环境，包括Python、Django、Vue和MySQL的安装与配置。
2. 部署后端服务，根据提供的文档和教程进行数据库配置与后端代码部署。
3. 部署前端服务，根据提供的文档和教程进行Vue项目的配置与运行。
